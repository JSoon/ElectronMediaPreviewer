# ElectronMediaPreviewer
基于 Electron 开发的多媒体消息预览器: 图片/视频

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
